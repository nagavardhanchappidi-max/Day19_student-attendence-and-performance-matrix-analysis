## Description
This project analyzes attendance and marks of 5 students using 2D arrays, generating attendance reports, marks reports, grade reports, and identifying toppers/lowest performers.

## Example Included
The menu-based system provides:
1. Attendance Report  
2. Marks Report  
3. Grades  
4. Highest/Lowest Performers
